# Финальный проект на тему "Арканоид"


## ⚙️ Функционал
- управление платформой (влево / вправо)
- движение и отражение мяча
- разрушение кирпичей
- подсчет очков
- система жизней
- экран победы и поражения
- перезапуск игры

## 🎮 Управление
- ← → — движение платформы  
- ПРОБЕЛ — запуск мяча  
- ENTER — начать игру  
- R — перезапуск  

## 🛠️ Технологии
- Python
- Pygame

## ▶️ Запуск проекта
Ввести в консоли: py -3.13 arkanoid.py
